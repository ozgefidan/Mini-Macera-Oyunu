# Mini Macera Oyunu (Java)

Bu proje, Java dili kullanılarak geliştirilmiş konsol tabanlı bir metin macera oyunudur. Oyuncu odalar arasında gezebilir, eşyalar toplayabilir, NPC’lerle konuşabilir ve oyunu tamamlamaya çalışır.

## Proje Özellikleri
- Oda tabanlı oyun haritası
- Kilitli kapılar ve anahtar sistemi
- Envanter ve can (HP) yönetimi
- Dost ve düşman NPC’ler
- Diyalog sistemi (conversation tree)
- Komut tabanlı konsol oynanışı

## Kullanılan Teknolojiler
- Java
- Nesne Yönelimli Programlama (OOP)
- Java Collections (List, Map, Set)
- Scanner ile kullanıcı girişi

## Oyun Mekanikleri
- Anahtar ile kilitli kapı açma
- Can iksiri ile sağlık artırma
- Silah kuşanma
- NPC’lerle etkileşim ve düşman saldırıları

## Komutlar
look → Odayı incele  
go [yön] → Hareket et  
take [eşya] → Eşya al  
use [eşya] → Eşya kullan  
talk [karakter] → Karakterle konuş  
inv → Envanteri göster  
help → Komutları listele  
quit → Oyunu bitir  

## Çalıştırma
javac MiniMaceraOyunu.java  
java MiniMaceraOyunu  

## Not
Bu proje eğitim amaçlı geliştirilmiştir ve OOP kavramlarını pekiştirmeyi hedefler.
